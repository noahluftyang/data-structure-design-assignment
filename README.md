## Prerequisite

- python >= 3.6.0
- nodejs >= 8.10.0

## Installation

Before install dependencies, you need to make virtual env first

```sh
  python -m venv .venv
```

And then install pip & npm packages!

```sh
  pip install -r requirements.txt
  npm install
```

## Development

```sh
  # run api server
  FLASK_DEBUG=1 python wsgi.py

  # run client app
  npm run develop
```
